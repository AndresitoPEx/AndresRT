Sobre mi

Mi especialización se centra en tecnologías como JavaScript, HTML y CSS. Tengo experiencia usando frameworks como React JS, React Native y .NET, además de librerias de estilos como Bootstrap, Tailwind CSS, Material UI y herramientas de diseño como Figma y Photoshop.
